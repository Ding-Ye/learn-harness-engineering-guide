learn-harness-engineering-guide

Re-implementing the Nexu Harness Engineering Guide in working Go, 14 chapters at a time.

What this repo is

The upstream Harness Engineering Guide is 13K lines of markdown that describes how to build an AI agent harness — agentic loop, tool registry, memory tiers, guardrails, sub-agents, classifier permissions, the works. This repo turns the prose into 14 progressive Go chapters, each ~150-700 LOC, each with tests, each pinned to specific upstream lines.

agentic loop → tool registry → memory → guardrails → retry → skills │ │ └─► context compression → event log → checkpoint → sub-agent ┘ │ cron scheduler → classifier permissions

Quick start

Clone the repo, run s01, look at the loop:

git clone https://github.com/Ding-Ye/learn-harness-engineering-guide.git cd learn-harness-engineering-guide/agents/s01-minimum-loop go test -count=1 ./... go run . "hello world" # I ran the echo tool on "hello world". Task complete.
